Welcome to the ELK Stack demo.

To jump straight in, go to http://127.0.0.1:5601 and configure the index pattern. Example data, visualizations, and an example dashboard should already be set up for you.

For a bit of background, check out the presentation to get an idea of how to use this. As this is part of a spoken presentation, it may not be as useful without the speaker notes. Press "s" when you get to that page to bring them up, which may help explain some of the slides a bit better.

Finally, if you want to play around with it, simply run "vagrant ssh" to SSH into the box in question. Modify config files, mess with settings, do what you want. Run "sudo salt-call state.highstate" in the VM, or "vagrant provision" outside of it, to bring things back to the installation state.

The example data provided is only good up to the point that 'vagrant up' was run, so after a few days this VM should probably be destroyed ('vagrant destroy') and re-upped ('vagrant up'). Have fun!
